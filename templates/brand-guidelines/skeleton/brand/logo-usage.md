# Logo Usage

Rules for using the __BRAND_NAME__ logo across all media and contexts.

## Primary logo

The primary logo is the full-color lockup combining the symbol and wordmark. Use this version whenever space and context permit.

### Minimum size

- **Digital:** 120px wide minimum
- **Print:** 30mm wide minimum
- Below minimum size, use the symbol-only mark

### Clear space

Maintain clear space equal to the height of the symbol on all four sides. No other visual elements, text, or edges should encroach on this area.

### Placement

- Prefer top-left or centered placement on pages and screens
- On dark backgrounds, use the reversed (white) version
- On photography or complex backgrounds, use the version with a background container

## Symbol mark

The standalone symbol (without wordmark) is used when the full logo is too small or when __BRAND_NAME__ is already identified by context.

- Minimum size: 24px digital, 8mm print
- Same clear space rules apply proportionally
- Use as favicon, app icon, and social media avatar

## Color versions

| Version | Use when |
|---|---|
| Full color | White or light neutral backgrounds |
| Reversed (white) | Dark or colored backgrounds |
| Monochrome (black) | Single-color printing, fax, stamp |
| Monochrome (white) | Single-color on dark surfaces |

## Logo misuse

The following modifications are not permitted:

- **Do not** change the logo colors to non-brand colors
- **Do not** rotate, skew, or distort the logo
- **Do not** add drop shadows, outlines, or effects
- **Do not** place the logo on busy backgrounds without a container
- **Do not** rearrange the symbol and wordmark
- **Do not** recreate the wordmark in a different typeface
- **Do not** crop or partially obscure the logo
- **Do not** animate the logo without brand team approval
- **Do not** reduce below minimum size requirements

## Co-branding

When the __BRAND_NAME__ logo appears alongside partner logos:

- Maintain equal visual weight — logos should appear approximately the same perceived size
- Use a vertical divider line (neutral-300) between logos with equal spacing on both sides
- __BRAND_NAME__ appears first (left in LTR, right in RTL) unless contractual agreements specify otherwise
- Each logo maintains its own clear space requirements

## File formats

| Format | Use case |
|---|---|
| SVG | Web, digital products, responsive contexts |
| PNG | Presentations, documents, social media |
| PDF | Print production, high-quality output |
| EPS | Legacy print workflows |

All logo files are maintained in the brand assets repository. Contact the brand team for access.
